# House Price Prediction

## Overview
This project focuses on predicting house prices using regression analysis techniques. It compares linear and non-linear models to understand how different features affect house prices.

## Models Used
- Multiple Linear Regression
- Spline-Based Regression (MARS-inspired)

## Dataset
The dataset used is the House Prices dataset, which includes features such as:
- Overall Quality
- Living Area
- Garage Capacity
- Basement Area
- Year Built

## Methodology
1. Data Preprocessing
2. Feature Selection
3. Model Training
4. Model Evaluation
5. Comparison of Results

## Results
The spline regression model captures non-linear relationships more effectively compared to linear regression in certain cases.
